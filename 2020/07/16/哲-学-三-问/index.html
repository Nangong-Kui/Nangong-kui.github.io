    哲 学 三 问
    发表于 2020-07-16| 更新于 2020-07-16
    阅读量:

    谨以此文，致敬存在主义先驱者——弗里德里希•威廉•尼采。

    叔本华说：

    “生命是一团欲望：
    欲望不满足，会痛苦。
    欲望若满足，会无聊。
    人生，就在痛苦与无聊之间摇摆。”

    那么唯一的不同是：

    穷人基本痛苦，富人大多无聊。

    娶 “痛苦”，还是嫁给 “无聊”？
    这，是一道你人生的必答题。

    如果你娶了“痛苦”：

    柴米油盐酱醋茶，日复一日年复一年。
    琐事缠身，痛苦至极。
    终于有一天你无法忍受，找了一个叫“文学”的小三。

    这时候萨特突然揪着你的耳朵大骂道：

    “当孩子快要饿死的时候，文学还有什么意义？”

    如果你嫁给“无聊”：

    琴棋书画诗酒花，日复一日年复一年。
    利欲熏心，无聊至极。
    终于有一天你无法忍受，找了一个叫“慈善”的情人。
    这时候罗素突然指着你的鼻子质问道：

    “如果连富人都不幸福，让所有的穷人变富又有什么意义？”

    于是，你夜里一个人躺在床上。

    翻来覆去，像一张摊在锅上的烙饼。

    “生命的意义”犹如笛卡尔的诅咒之火，无时无刻不在炙烤着你失眠的灵魂。

    你只好吞下一颗安眠药，无效。
    又吞了一颗，依然。
    你抓起一把全部吞下，终于进入了梦乡。

    你躺在一片荒野里，身边围了一圈穿白色衣服的树。
    你坐起身，向远处望去。
    一道黑色闪电顺着你的目光，从地上犁出一条河。

    你似乎想起了什么，于是死死地抱住了一颗树，漂到了一个叫“哲学岛”的地方。

    你下了船，走上岛。

    正好遇见了披着袈裟的释迦牟尼，你问：“什么是佛法？”

    释迦牟尼说：“诸法无所有，空拳诳小儿。”

    你失望了，转身看到骑着青牛的老子，又问：“什么是道？”

    老子说：“道可道，非恒道。”

    你更加痛苦，跑到十字架上的耶稣面前，继续问：“什么是上帝？”

    耶稣回答道：“信者上天堂，不信者下地狱。”

    “一群骗子！”

    你忍不住破口大骂。

    “世界上既然没有佛法，没有道，也没有上帝。
    生命既然没有任何意义，你们为什么要欺骗世人？”

    你的质问声惊动了苏格拉底。

    苏格拉底走到你身边问道：

    “如果你看到一只美丽的梅花鹿跑进森林，这时候一个猎人向你询问梅花鹿的行踪，你选择诚实还是欺骗？”

    你准备回答却又哑口失言，沉思了片刻你恍然大悟：

    如果诚实需要生命作为代价，诚实还有什么意义呢？

    于是你欣然选择欺骗，离开了小岛，回到俗世。

    直到有一天你突然发现，原来这只是苏格拉底的诡辩。

    欺骗何尝没有代价？

    当所有人都选择欺骗的时候，当梅花鹿知道猎人这个死神永远也无法找到它的时候，梅花鹿便堕落了。

    于是，你再次陷入了沉思。

    堕落地活，还是有尊严的地死？
    这，又成了一道你人生的必答题

    你茫然地在海边游荡，突然被一位正在遛狗的哲学家撞倒，

    他并没有去扶你，而是瞪着坐在地上的你扔下一句：

    “学习哲学，就是学习如何有尊严地去死！”

    你望着他远去的背影，无地自容。

    你屏住呼吸，终于想起这是蒙田随笔录当中的一句。
    你站起身，走上一片断崖。

    望着眼前的大海，仿佛看到了亚里士多德，看到了他因无法理解潮汐而跳海的身影。你也同样无法理解生命，或许这就是命。

    你慢慢地闭上了眼睛，身体微微前倾。

    这时黑格尔正好路过，他冲着你的背影大声劝道：

    “生活的真谛在于活着，好好活着便是人生的意义。”

    你开始犹豫。

    康德恰巧也在崖下散步，抬头一看，厉声骂道：

    “自杀是一种对自我的可耻谋杀，你居然把自己看成了一个可以被任意宰杀的动物。你以为自杀意味着解脱？错！只有活着才能感受解脱之味，死亡会让你失去知觉。”

    你如梦方醒，于是睁开眼睛，缓缓地走下石崖，回到海边。
    你已经无法记起这是你第几次踏入海水了，因为每一次都是同一种感觉。
    难道是赫拉克里特欺骗了我？还是我的感觉欺骗了我？抑或是因为我们踏入的不是同一条河？”

    你正在思索，突然看到不远处有一位大胡子先生正在慷慨地演说。

    你走过去拨开人群，高声问道：
    “安提丰，人活着到底有什么意义？”

    安提丰顿了一下，耸了耸肩说：

    **“我至今也没有弄清楚，所以我要活下去。”

    周围一片笑声，唯有你低头不语。**

    你决定重新出发去寻找生命的意义，你刚转身就看到海边有一条小船，船上钉坐着一位垂钓的老者。

    你上了船问道：“能带我去哲学岛吗？”
    老者回答说：“不用去了，哲学岛已经沉没，我刚从那儿逃回来。”

    你惊愕地看着他，然后问：“既然你是岛上的哲学家，那你告诉我生命究竟有什么意义？”

    老者回答道：“日方中方睨，人方生方死，一受精而成其形，不亡以待时尽。”
    你顿时如醍醐灌顶，终于听到了哲学家的真言：

    原来人生，不过是在等死而已。

    于是你日日笙歌，夜夜买醉。

    仿佛帕斯卡尔在你的内心中呐喊：

    “把生命当做只剩八天那样来狂欢吧。”

    及时行乐的你，像一只嬉戏在屠夫眼皮下的羔羊，随时待宰。

    但是第九天，死神没有如期而至。

    你开始怀疑、迷茫、空虚。
    你突然感到一种巨大的、无法言说的失落。
    你知道，这不是真正的快乐。
    这，不是你想要的生活。

    然后你开始每天独自一人，朝着日出的方向，眺望大海。
    你不知道自己在眺望什么，但是，你知道自己必须眺望。

    突然有一天，你看到一艘小船在海浪中穿行。

    银白如雪，轻快如鱼，然后被海浪打翻，你轻蔑地一笑。

    第二天，你又看到了那一艘小船，再次被浪打翻。

    第三天，还是如此。

    第四天，你忍不住向海边的一位渔夫打听：“驾船的人是谁？他为什么每天出海去找死？”

    渔夫说：“他就是个疯老头，疯了一辈子，你没看见他的船上写着七个字吗？”

    “什么字？”

    “知其不可而为之。”

    你愣在原地，思索良久。

    突然放声狂笑，笑到听不见自己的心跳，笑到时间仿佛静止。

    然后瘫坐在地上，失声痛哭。

    第五天，当你又看到那一艘小船在海浪中穿行时。

    仿佛听到了大海在鸣唱，那是一种激荡而低沉的声响。

    你仔细听，没错，大海正在演奏着一曲生命的交响乐。

    那一刻你终于发现：

    那一艘飘摇的小船，原来是一个人的灵魂在舞蹈。
    只是，在那些听不到音乐的人眼里，跳舞者都是疯子。

    那一刻你终于明白：

    真正的勇者，绝不会欺骗自己。

    纵然生命没有意义，他也要载着这无意义的生命，载着这无处安放的灵魂，勇敢地驾船航行。

    勇者的生命不需要让“意义”去替他欺骗死神，相反，他会勇敢地告诉死神，他的灵魂所在。然后静待死神追来，跟死神赛跑、捉迷藏、斗智斗勇。

    那一刻开始，你发誓要做一个“超人”。

    超越过去的自己，超越那些终生待在岸上徘徊的懦夫们。
    后来，死神在你思想的森林中迷了路。
    于是很久很久，你都无法找到对手。

    终于有一天，孤独把你逼上了绝路，就像把一条河流逼下悬崖。

    你的精神崩溃，散落成了瀑布。

    你思考的脚步停止了，但是你的思想却长出了翅膀，变成一只飞翔的鹰。
    依然在哲学的天空上，代替你追寻。
    鹰飞的越来越高，越来越远，在世人的眼里，却越来越渺小。

    终于，鹰也消失在了人们的视线里，它飞过海洋，飞过森林，飞过雪山，飞到了一片世人无法企及的遥远之地，最后降落到一块墓碑上。

    墓碑上刻着：弗里德里希•威廉•尼采。

    这一刻，你的梦醒了。
    泪水浸湿了枕头，你艰难地爬起身。
    照了照镜子，微笑和阳光都在。

    你，还活着。

    形而上学思考
